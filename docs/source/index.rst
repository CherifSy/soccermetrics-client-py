Support and Development
=======================

All development occurs at `GitHub <https://github.com/soccermetrics/soccermetrics-client-py/>`_.
To checkout the source:

.. sourcecode:: bash

   $ git clone git@github.com:soccermetrics/soccermetrics-client-py.git dir_name

Report bugs with the
`GitHub issue tracker <https://github.com/soccermetrics/soccermetrics-client-py/issues>`_.

We welcome all contributions to this project.  Before you do contribute, please read our
`contribution guidelines <https://github.com/soccermetrics/soccermetrics-client-py/blob/master/CONTRIBUTING.rst>`_ first.

If you have any questions about the client and its use in your applications, please seek help in the following order:

- The documentation for the `API client`_ and the `API`_ itself.
- The `Developer Forum`_ at the Soccermetrics API Developer Portal.
- Message to our `Twitter feed <http://twitter.com/sccrmetricsapi>`_
- Message to our support email address (api-support@soccermetrics.net).
