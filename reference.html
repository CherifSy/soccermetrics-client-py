class soccermetrics.rest.SoccermetricsRestClient(account=None, api_key=None, base_uri='http://api-summary.soccermetrics.net')[source]¶

A client object for accessing the Soccermetrics REST API.

Parameters:
  • account (string or None) – Soccermetrics API Application ID.
  • api_key (string or None) – Soccermetrics API Application key.
soccermetrics.rest.find_credentials()[source]¶

Search for API credentials in current environment.

Looks for SOCCERMETRICS_APP_ID and SOCCERMETRICS_APP_KEY among the environment variables. Returns a KeyError if either variable is not present in the environment.

Returns:(account, api_key) or (None, None)
